# HEYGEN VIDEO PRODUCTION — MANUAL GUIDE

## For each agent (11 total):

1. Go to app.heygen.com → Create Video

2. Choose 'Photo Avatar' → Upload the agent's avatar PNG

3. Select Avatar IV engine

4. For voice: Upload the reference WAV → Clone Voice

5. Paste the script text below into the script box

6. Set background color (listed per agent)

7. Generate → Download as MP4

8. Save as: {agent}_intro.mp4
